Benutzer

Es gibt zwei verschiedene Benutzerklassen. Auf der einen Seite gibt es Berufsbildner/innen, diese haben die Rolle der Administratoren und können deshalb Benutzer und Tasks verwalten. Zudem können sie Produkte im Shop erstellen. Auf der anderen Seite gibt es die Lernenden. Diese können keine Tasks oder Benutzer verwalten, jedoch können sie Tasks erledigen und einreichen.

Tasks

In der Taskverwaltung können Berufsbildner/innen Taks erstellen und einer/einem Lernenden oder gleich mehreren zuweisen. Die Lernenden sind nach Lehrjahr sortiert und können so einfach ausgewählt werden. Die Lernenden können die Tasks, welche ihnen zugewiesen sind einsehen und diese einreichen. Wenn ein/e Lernende/r ein Task einreicht, kann ein/e Berufsbildner/in überprüfen ob dieser tatsächlich abgegeben wurde. Wenn dies so ist, kann sie/er ihn akzeptieren und die/der Lernende/r erhaltet die Coins. Andernfalls erscheint der Task wieder als nicht erledigt bei der/dem Lernende/r.
